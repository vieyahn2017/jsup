# 生辰小助手
一个实用的生辰八字计算小工具，输入生辰日期时间，然后自动计算出八字，五行等信息。

[点击这里](http://jamling.coding.me/birthday-tool/)体验吧。

# 注意
- 输入的出生日期需精确到小时
- 有效的时间段为1901到2100年，超过此时间段，本山人也算不出来啦。
- 旧版是以阴历正初一做农历年的开始（并不准确）
